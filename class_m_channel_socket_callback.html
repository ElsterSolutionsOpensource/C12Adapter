Detailed Description

Socket callback channel is MChannelSocket that has Auto Answer enabled by default.

This class is a convenient way of establishing socket servers, however one has to remember this class can only handle one request at a time.

Different from MChannelSocket, this class sets persistent property MChannelSocket::SetAutoAnswer to true by default. This property can be set to false by applications, in which case the behavior of this class will be indistinguishable from its parent.
